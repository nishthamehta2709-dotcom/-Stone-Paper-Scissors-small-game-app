
# 🎮 Stone Paper Scissors Game (Python)

A fun and interactive **Stone-Paper-Scissors game** built using Python. This project includes a simple UI, sound effects, and game logic to make it feel like a real mini game.

---

## 📌 Features

* 🧠 User vs Computer gameplay
* 🎨 Simple and attractive UI (Tkinter)
* 🔊 Sound effects (click, win, lose)
* 🎲 Random computer choice
* 📊 Score tracking system
* ⚡ Fast and responsive gameplay

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (for GUI)
* Random module
* Playsound / Pygame (for audio)

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/stone-paper-scissors.git
```

2. Navigate to the folder:

```bash
cd stone-paper-scissors
```

3. Install required libraries:

```bash
pip install pygame
```

4. Run the game:

```bash
python main.py
```

---

## 🎮 How to Play

* Choose **Stone 🪨**, **Paper 📄**, or **Scissors ✂️**
* The computer will randomly select its move
* The winner is decided based on rules:

  * Stone beats Scissors
  * Scissors beats Paper
  * Paper beats Stone
